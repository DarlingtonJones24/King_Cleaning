# King Cleaning Website

This folder contains the King Cleaning B.V website built with Next.js.

## Start the website

Open a terminal in this folder:

```bash
C:\Users\user\Desktop\Cleaning BV\CleaningBV\cleanzy-next
```

Install the project packages:

```bash
npm install
```

Start the website:

```bash
npm run dev
```

Then open this link in your browser:

```bash
http://localhost:3000
```

If PowerShell blocks `npm`, use this instead:

```bash
npm.cmd install
npm.cmd run dev
```
